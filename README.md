## TinyHAP是什么?

    TinyHAP是一个跨平台的HomeKit实现, 开发语言使用标准C语言开发(C99).

    支持如下平台:

    1. Windows(x86/x64), 标准Windows平台.
    2. Linux(x86/x64), 绝大多数Linux发行版.
    3. Linux(ARM), 各种基于ARM的嵌入式Linux平台.
    4. MacOS(x64), 苹果iMac/MacBook/MacBookPro等.
    5. FreeRTOS(ESP8266), 乐鑫的wifi模组平台.

    不依赖任何第三方库.
